## Naming Convention

Format: `NNN-abv-title.ext`

- NNN: Zero-padded 3-digit number (000-999)
- abv: 3-letter category abbreviation
- title: Kebab-case description
- ext: File extension

### Category Abbreviations
- idx: Index/directory
- ref: Reference documentation
- sec: Security
- dev: Development
- sch: Schema
- pmt: Prompt
- map: Mapping/structure
- ana: Analysis
- rel: Release
- ops: Operations/DevOps
- int: Internal/infrastructure
- chg: Changelog
